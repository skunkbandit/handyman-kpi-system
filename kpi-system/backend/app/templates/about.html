System Overview

The Handyman KPI System is a comprehensive performance tracking and evaluation platform designed specifically for handyman businesses with tiered skill progression.

This system allows managers to track employee skills, tool proficiency, and performance metrics over time, providing valuable insights for training, advancement, and business planning.

Key Features
  • Digital skill evaluation forms with predefined categories
  • Tool proficiency tracking
  • Employee tier progression management
  • Performance dashboards with data visualization
  • Historical performance tracking
  • Reporting and export capabilities
Tier System
Tier Description

Rating System

The system uses a 1-5 rating scale for evaluating employee skills:

Rating Description Visual Representation
1 Never performed this skill
2 Performed once or twice
3 Need a bit more training
4 Can do it but not a pro
5 I am a DAMN PRO

For tools, the system tracks two aspects:

  • Can Operate: The employee knows how to use the tool properly and safely
  • Owns Tool: The employee owns this tool as part of their truck stock
